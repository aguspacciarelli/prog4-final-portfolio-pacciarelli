Swing Club

El proyecto

Swing Club nació como un proyecto académico de la materia Programación Multimedial III, con el objetivo de desarrollar un e-commerce para los aficionados de la música en su versión analógica. Este e-commerce está inspirado en el jazz, el blues, el soul, el góspel y el R&B. Se venden productos como vinilos, cassettes, cds, tocadiscos, accesorios para tocadiscos e instrumentos, entre otras cosas similares.

El problema parte de que hoy en día el formato analógico musical no solo no cuena con los algoritmos para recomendarnos este tipo de productos en la vida real, sino que no hay centralización a la hora de comprar, por lo que se hace tedioso encontrar gran cantidad de productos que deseamos en un mismo lugar sin que varíe demasiado el precio entre tiendas.

Objetivos de Swing Club

Ofrecer una experiencia premium, personalizada e innovadora en la compra de productos de música analógica, incluyendo aquellos difíciles de encontrar, fomentando al mismo tiempo el resurgimiento y la apreciación de este formato.

¿Qué necesidad tiene el público que SwingClub puede resolver?

SwingClub le facilita a sus usuarios a conseguir en un mismo lugar, no sólo seguro sino actualizado y fácil de usar, artículos de música analógica. Estos son productos que no son fáciles de encontrar en todos lados, sobre todo siendo productos de nicho. SwingClub también tiene un sistema de recomendaciones para aquellos que quieran ampliar su conocimiento y gusto musical y no saben por dónde empezar, sumado a recomendaciones de productos que les podrían interesar en base a este sistema.

Identidad visual

La propuesta del logo y la identidad de Swing Club busca atraer con sus colores vibrantes, con una influencia de Swiss Style, cubismo y una estética artesanal.

Átomos, íconos y más!

El UI Kit de Swing Club con el cual se construyó la base para diseñar y desarrollar el sitio web del e-commerce. Cards de producto, carrito, filtros y navegación diseñados para una experiencia de compra fluida.

Resultado :)

Aplicación real de un e-commerce funcional y coherente, con la marca aplicada de forma consistente en todo el flujo de compra.
